### What is software testing?
Software testing is a process of executing a program or application with the intent of finding any errors, gaps, or missing requirements in contrast to the actual requirements.

### What is the difference between software testing and debugging?
Software testing is a process of executing a program or application with the intent of finding any errors, gaps, or missing requirements in contrast to the actual requirements.
Debugging is a process of finding and fixing errors in a program or application.

### What is the difference between software testing and quality assurance?
Software testing is a process of executing a program or application with the intent of finding any errors, gaps, or missing requirements in contrast to the actual requirements.
Quality assurance is a process of ensuring that the software meets the requirements and is of high quality.

### How many types of software testing are there?
There are two types of software testing:
1. Manual Testing
2. Automation Testing

### What is manual testing?
Manual software testing is the process of manually checking software for bugs, defects, or issues without using any automation tools. A human tester plays the role of an end user and tests all the features of the application by clicking, typing, and navigating through the app to ensure it behaves as expected.
